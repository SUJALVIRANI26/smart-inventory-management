Required Fields

  • sku (Required, Unique)
  • category (Required)
  • price (Required, Number)
  • quantity (Required, Number)
  • brand (Optional)
  • description (Optional)
  • cost_price (Optional)

Important Notes

  • File must be in CSV format (UTF-8 encoding recommended)
  • Maximum file size: 10MB
  • Maximum records per import: 5000 products
  • Existing products with same SKU will be updated
  • New products with unique SKU will be created
  • All monetary values should be in USD
